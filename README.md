# 📈 Indian Stock Market Dashboard

An interactive dashboard built with **Python**, **Streamlit**, and **NSELib** that allows users to explore various datasets from the Indian Stock Market through an easy-to-use interface.

---

## 🚀 Features

- Interactive Streamlit dashboard
- Multiple NSE market datasets
- Dynamic sidebar options
- Date and period-based data fetching
- Clean and responsive UI
- Displays data in tabular format

---

## 📊 Available Data

- Bhav Copy
- Bhav Copy with Delivery
- Equity List
- F&O Equity List
- Nifty 50 List
- Market Watch
- India VIX
- Block Deals
- Bulk Deals
- Deliverable Position Data
- Price Volume Data
- Price Volume & Deliverable Position Data
- Short Selling Data
- Index Data

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Indian-Stock-Market-Dashboard.git
```

Go to the project folder

```bash
cd Indian-Stock-Market-Dashboard
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate the environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run dashboard.py
```
